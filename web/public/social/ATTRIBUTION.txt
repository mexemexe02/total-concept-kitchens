Total Concept Kitchens — social media stills (local copies)

Source: Instagram @totalconceptkitchens (Meta CDN, scontent-*.cdninstagram.com).
Captured: 2026-04-05 via authenticated web session for use on this marketing site only.

Files:
- hero-ig.jpg — hero background
- ig-01.jpg … ig-06.jpg — gallery grid fallbacks when the Instagram Graph API is not configured

These URLs expire on Meta’s CDN; local files stay stable for Docker/Coolify builds.
Replace or add images anytime; keep this file updated if you swap assets.

If these files are missing from the repo clone, the site uses Unsplash sample kitchens
for `/gallery` and the hero (see `src/lib/gallery-fallback-remote.ts`) until you restore
JPEGs here or configure `INSTAGRAM_ACCESS_TOKEN` + `INSTAGRAM_USER_ID`.

Caption text under gallery tiles is copied from the matching Instagram posts (see
web/src/lib/gallery-fallback.ts and web/content/business-profile-source.json).

hero-ig.jpg must stay a different file from ig-01.jpg … ig-06.jpg (hero is not repeated
in the gallery). The app throws at startup if a path is duplicated in the fallback list.
