Official Total Concept Kitchens logo
-------------------------------------
`logo.png` in this folder is the brand mark (black + cyan emblem + wordmark).
Use a PNG with a transparent background so it sits cleanly on the cream / charcoal header
(no white rectangle behind the mark).

To replace it later: overwrite `logo.png` (or add `logo.jpg` / `.webp` — the header tries
extensions in order; see `LogoOrWordmark.tsx`).

If no logo file is present, the header falls back to the “TCK” text wordmark.
